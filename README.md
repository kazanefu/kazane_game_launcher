# kazane_game_launcher

## インストール方法

[最新バージョンのリンク](https://github.com/kazanefu/kazane_game_launcher/releases/latest)

1. 上記のリンクからインストール`kazane_game_launcher.exe`をクリックしてダウンロードします。
2. kazane_game_launcher.exeがダウンロードのところにあるはずなので適当な場所に移動させてください(初回起動時に設定ファイルやインストール済みのゲーム情報などのファイルを生成するため専用のフォルダを準備しておくことをおすすめします)
3. kazane_game_launcher.exeを起動します。もし警告がでたら**詳細**をクリックして**実行**を選択してください
4. これで初回起動完了です。

## 使い方

### ゲームのインストール

1. 左上で`Search`を選択します
2. `Search`の枠にゲーム名, `Tags`にタグを入力して検索することができます。何も入れずに検索ボタンを押すとすべてのゲームを表示させることができます
3. 検索をして出てきたものの`README`ボタンを押すとREADMEを表示させることができます
4. Installボタンを押すとインストールされます。インストールされるゲームは設定ファイルで指定した場所にインストールされます(デフォルトではkazane_game_launcher.exeから相対パスで`games/`にインストールされます)
5. Uninstallボタン/Updateボタン/Launchボタンが表示されるようになったらインストール成功です

### ゲームの起動

1. 左上で`Library`を選択します
2. インストール済みのゲーム一覧が表示されているので、起動したいゲームの`Launch`ボタンを押すことで起動できます

### ゲームのアンインストール方法

1. Library or SearchでアンインストールしたいゲームのUninstallボタンを押します

### うまくいかないとき

もし何かうまくいかないことがあったら`Logs`を確認して私に報告してください

### 設定ファイルについて

初回起動後に`kazane_game_launcher.exe`から相対パスで`launcher/settings.json`に設定ファイルが生成されます。

デフォルトでは以下のようになっています
```json
{
  "install_dir": "games",
  "theme": "dark",
  "size": 1.0
}
```
- "install_dir"の欄には`kazane_game_launcher.exe`からの相対パスでゲームのインストール先を指定します
- "thema"にはテーマを"dark"か"light"から選択してください
- "size"にはUIのサイズを指定します

## その他ファイル

- `local/game_data.json`: インストール済みゲームの情報が書いてあるファイルです。基本的にはいじらないでください。
- `local/runtime.log`: 今までの操作のログを保存しているファイルです。もしバグとかが起きた場合はそのファイルを私に送っていただけるとありがたいです。
